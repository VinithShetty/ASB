Configuration Options

1. Custom Success/Error Messages

formData.append("redirect", "https://yourdomain.com/thank-you.html");

2. Email To (if not set in dashboard)

formData.append("email_to", "sales@asbengineeringservices.com");

3. Reply-To Email

formData.append("replyto", formData.get("email"));

4. Custom Subject with User Input

formData.append("subject", "New Quote Request - " + formData.get("subject"));

5. Add Hidden Fields for Tracking

formData.append("page_url", window.location.href);
formData.append("page_title", document.title);
formData.append("user_agent", navigator.userAgent);
